## Overview  
This repository contains my structured documentation and technical writeups of Capture The Flag (CTF) challenges and cybersecurity labs completed across various platforms.  

The purpose of this repository is to:  
- Strengthen my offensive security skills  
- Document my learning process   
- Develop professional technical reporting habits  
- Track my cybersecurity journey  

### Platform Covered  
- TryHackMe

 ### Repository Structure 
 Write-ups/  
│  
├── TryHackMe/  
│   ├── Linux-Fundamentals/    
│   ├── ..  
│   
└── README.md  

#### Each room or machine contains its own README.md with:  

- Objective  
- Enumeration steps  
- Exploitation process  
- Privilege escalation (if applicable)  
- Security analysis  
- Key takeaways

#### Rather than simply solving challenges, I focus on:  
- Understanding why an attack works  
- Analyzing root causes of vulnerabilities    
- Relating concepts to real-world security scenarios  
- Writing structured technical reports
