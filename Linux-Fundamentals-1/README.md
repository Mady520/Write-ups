## Linux Fundamentals 1 – TryHackMe

### Overview  
This room provided foundational knowledge of the Linux operating system, focusing on command-line usage, file system navigation, permission management, and basic system administration concepts.  
Linux is widely used in servers, cloud infrastructure, and cybersecurity environments. Mastering Linux fundamentals is essential for penetration testing, incident response, and system security analysis.  

### Objectives  
Understand the Linux file system structure.  
Navigate directories using the command line.  
Manage files and directories.  
Learn file permissions and ownership.  
Understand basic user and process management.  
Develop confidence using the terminal.  

### Commands practiced  
```bash
echo

```
### Concepts learned  
Identifying current user  
Listing directories  
Changing directories/folders/files  
Reading files/system   
Navigating directories efficiently  
Finding a particular thng in files/folders  

### Shell operator commands

```bash
&
&&
>
>>
```
### Concepts learned  
Run commands/functions in the background of terminal.  
Combine multiple commands together in one line.  
Take output of one command and send to somewhere else.  
Append the output to the bottom of the file.
